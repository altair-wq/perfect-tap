# v2 Ideas — Parking Lot

Everything that would be cool but is NOT in v1. Revisit only after v1 ships.

## Quick wins (low effort, high impact)
- Daily challenge: deterministic seed per day, shared bar speed/zone pattern, screenshot leaderboard
- Skins: 5–10 color themes unlocked at score milestones (cosmetics only, no power)
- Particle effect variants per skin
- Streaks tracker: "You played 7 days in a row"
- Share button on game over: auto-generate a clean "I scored X" image

## Medium effort
- Endless mode + Levels mode (handcrafted patterns, 50 levels)
- Two-bar mode: tap when both align
- Audio mode: bar invisible 50% of cycle, tap to the beat
- Hold-and-release variant
- Local 2-player pass-and-play

## Bigger bets
- Online leaderboard (needs backend — Supabase free tier is fine)
- Async PvP: your ghost run challenges a random player
- Rewarded ad for "revive once per run"
- IAP: $2.99 remove ads, $0.99 per skin pack

## Game variants to spin off
- "Score Hero Hockey" — the swipe-pass puzzle we discussed
- "Perfect Putt" — golf swing meter with course mode
- "Beat Tap" — synced to royalty-free music tracks
